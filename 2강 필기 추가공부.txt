C++에서 strcmp() 함수를 사용하여 두 개의 문자열이 같은지를 확인하려면, 함수가 반환하는 값을 if 문과 함께 사용하면 됩니다. strcmp() 함수가 0을 반환하면, 두 문자열이 서로 같다는 것을 의미합니다. 따라서, 다음과 같은 코드를 사용하여 두 개의 문자열이 같다는 것을 확인할 수 있습니다.

c++
Copy code
if (strcmp(str1, str2) == 0) {
    // 두 문자열이 같은 경우
}
위 코드에서 str1과 str2는 비교하려는 두 개의 문자열입니다. strcmp() 함수가 반환하는 값이 0이면, if 문 안의 코드가 실행됩니다. 따라서, 두 개의 문자열이 같은 경우, if 문 안의 코드가 실행되도록 할 수 있습니다.


1을 반환하면 같지 않다는 뜻인가? 그러면?
네, 맞습니다. strcmp() 함수가 0이 아닌 값을 반환하면, 두 개의 문자열이 다르다는 것을 의미합니다. 따라서, strcmp() 함수가 0이 아닌 값을 반환하면, 다음과 같이 두 개의 문자열이 다르다는 것을 확인할 수 있습니다.

c++
Copy code
if (strcmp(str1, str2) != 0) {
    // 두 문자열이 다른 경우
}
위 코드에서 str1과 str2는 비교하려는 두 개의 문자열입니다. strcmp() 함수가 반환하는 값이 0이 아니면, if 문 안의 코드가 실행됩니다. 따라서, 두 개의 문자열이 다른 경우, if 문 안의 코드가 실행되도록 할 수 있습니다.
